<Section icon={Map} title="Oofファーミングの基本">

Noob Incrementalは、Noobの軍隊を管理して「Oof」を蓄積することを主な目的とする、階層型Robloxアイドルゲームです。新しいプレイヤーにとって、最初の30分は非常に重要です。新しいNoobの獲得コストとアップグレードの指数関数的な価格のバランスを取る必要があります。他のインクリメンタルゲームとは異なり、Noob Incrementalは複数のシステムを同時に通してスケールするため、中盤の壁を乗り越えるために単一のリソースに頼ることはできません。

**ビデオハイライト:**
- **初期設定**: 最初の数体のNoobを購入して、Oof生産サイクルを開始します。
- **アップグレードの優先順位**: どの序盤のショップアイテムが最も高い投資収益率を提供するかを特定します。
- **ルーンの紹介**: ルーンの厳選がアカウントに永続的なスケーリングのレイヤーを追加する方法を初めて見ます。

<Section icon={Sparkles} title="ルーンシステムの習得">

Oofの安定した流れを確立したら、ルーンシステムが次の主要な焦点となります。ルーンは、厳選してレベルアップすることで、ステータスに巨大で永続的なボーナスを提供するユニークなアイテムです。このシステムは、「ルーン運」と「ルーン速度」に大きく影響され、これらは両方ともコミュニティエンゲージメントを通じて強化できます。

ルーンシステムは、長期的な進行メカニズムとして設計されています。特定の資源を使用してルーンを厳選し、各ルーンにはティアがあります。高ティアのルーンはより良いステータスブーストを提供しますが、見つけるのは著しく困難です。ここで、コミュニティボーナスが真剣なプレイヤーにとって不可欠になります。

| ボーナスタイプ | 出典 | 効果 |
| :--- | :--- | :--- |
| **ルーン速度** | **グループ + いいね** | より速い厳選セッションのための**+10% ルーン速度**。 |
| **ルーン運** | **Discord** | より良いティア結果のための**+10% ルーン運**。 |
| **Oofボーナス** | **Roblox Plus** | アクティブ中の**+10% Oof**生成。 |
| **プリズムボーナス** | **Roblox Plus** | レイトゲームスケーリングのための**+10% プリズム**獲得。 |

<Callout type="info" title="コミュニティ設定">
効率を最大化するには、Ghoulax Games Robloxグループに参加し、ゲームに「いいね」をしていることを確認してください。次に、コミュニティサーバーでDiscordアカウントをリンクします。これらのアクションは無料で、ルーンの進行に永続的な合計20%の効率ブーストを提供します。
</Callout>

ルーンのレベルアップは、ルーンを見つけることと同じくらい重要です。重複するルーンや特定のレベルアップ素材を集めることで、既存のルーンをアップグレードしてその効力を高めることができます。これにより、「コモン」ルーンであっても、中盤に移行しても有用なままになります。

| ルーンアクション | プレイヤー価値 | 最適な使用例 |
| :--- | :--- | :--- |
| **厳選** | 新しいステータスブーストを発見する | 序盤のアップグレードが高すぎると感じるとき。 |
| **レベルアップ** | 現在のボーナスを増やす | 特定のプレステージ要件を突破するため。 |
| **装備** | ステータス変更をアクティブにする | 常に最高ティアのルーンをアクティブにしておく。 |

<Callout type="warning" title="運のポーション">
コードからルーン運ポーションを入手した場合、すぐに使用しないでください。中断せずにできるだけ多くのルーンを厳選する時間が少なくとも15〜20分ある「プッシュセッション」のために保存しておきましょう。
</Callout>

</Section>

<Section icon={Trophy} title="プレステージレイヤーの攻略">

プレステージは、強力で永続的な乗数と新しいコンテンツへのアクセスと引き換えに、現在の進行状況をリセットできるメカニズムです。Noob Incrementalには複数のプレステージレイヤーがあり、それぞれがより深いレベルの複雑さを提供します。いつプレステージするかを理解することが、トップティアのプレイヤーと「初心者」の違いです。

<Steps>
  <Step num="1" title="基本ランを構築する">
    NoobとOofの生産に完全に集中します。アップグレードのコストが数百万になるまで、ルーンやプレステージについては心配しないでください。
  </Step>
  <Step num="2" title="強化アップグレードを購入する">
    利用可能なショップアップグレードを使い果たします。「次のアップグレード」の価格が現在のOPSの100倍になったら、ソフトキャップに達したことになります。
  </Step>
  <Step num="3" title="ルーンの進行を追加する">
    数分間ルーンを厳選して、追加の20〜50%のパワーを絞り出します。これにより、プレステージボタンが表示されるために必要な最後のプッシュが提供されることがよくあります。
  </Step>
  <Step num="4" title="ルーンボーナスをアクティブにする">
    グループとDiscordのボーナスがアクティブであることを再確認してください。これにより、プレステージ後のルーンの厳選が可能な限り効率的になります。
  </Step>
  <Step num="5" title="プレステージオプションを確認する">
    プレステージ報酬を確認します。リセットが「新しいコンテンツのアンロック」または全体的な成長に少なくとも5倍の乗数を提供する場合、それがその時です。
  </Step>
  <Step num="6" title="プレステージして再構築する">
    ボタンを押します。OofとNoobは失われますが、ルーンと永続的なアップグレードは保持されます。すぐにNoobを購入し始めて、以前の速度に戻しましょう。
  </Step>
  <Step num="7" title="新しいレイヤーを通して繰り返す">
    各プレステージレイヤーは新しいステータスを導入します。2回目または3回目のリセット後に表示され始める「プリズム」やその他のレイトゲームリソースに注意してください。
  </Step>
</Steps>

<Callout type="warning" title="早期プレステージのリスク">
早すぎるリセットはよくある間違いです。プレステージ報酬が1.2倍または1.5倍の乗数しか提供しない場合、再構築に費やす時間が節約した時間よりも長くなる可能性が高いです。序盤のアップグレードをどれだけ速く購入できるかを大幅に変える乗数を目標にしましょう。
</Callout>

</Section>

<Section icon={Target} title="無料報酬とコードの最大化">

コードは、**Noob Incremental 初心者ガイド**戦略の重要な部分です。これらは、新しいプレイヤーが最初の作業の最も遅い部分をスキップできる「キャッチアップ」メカニズムを提供します。ほとんどのコードは、チケット、トークン、ポーションの組み合わせを提供します。

コードを引き換えるには、画面下部の**ショップ**アイコンを開く必要があります。一番下までスクロールして、**ABXセクション**に移動します。ボックスにコードを入力し、「引き換える」を押します。コードが成功すると、報酬はすぐにインベントリに追加されます。

| アクティブコード | 報酬タイプ | カテゴリ |
| :--- | :--- | :--- |
| **THANKS4JOINING** | 混合 (トークン/チケット/ポーション) | コミュニティ報酬 |
| **RELEASE** | 混合 (トークン/チケット/ポーション) | リリース記念 |
| **WWWRELEASE** | 混合 (トークン/チケット/ポーション) | リリース記念 |
| **BUGFIX1** | 10 チケット | メンテナンス報酬 |
| **1.5KCCU!!** | ポーション | マイルストーン報酬 |
| **3KCCU!!** | 混合 (トークン/チケット/ポーション) | マイルストーン報酬 |

<Callout type="success" title="引き換え成功">
大文字と小文字の区別によるエラーを避けるため、常にコードを直接コピー＆ペーストしてください。ほとんどのNoob Incrementalコードは大文字で、「!!」のような句読点が含まれている場合があり、これらはコードが機能するために含める必要があります。
</Callout>

チケットとトークンは賢く使うべきです。即座のOofブーストを購入したくなるかもしれませんが、これらの通貨を永続的な「ゲームパス」や、すべてのプレステージレイヤーを通して持続する高ティアのルーンモディファイアのために貯めておく方が良いことがよくあります。ただし、ポーションは、10分間の集中したアクティブプレイセッションを計画しているときにいつでも使用すべきです。

</Section>
