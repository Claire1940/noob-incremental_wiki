<Callout type="info" title="クイックガイド">
- **Noob Incrementalのルーンレベル**は、パッシブボーナスとリソース生成のスケーリング強度を決定します。

<Section icon={Map} title="ルーンシステムの概要">

Noob Incrementalの2026年メタにおいて、ルーンはプレイヤーが利用できる最も重要なスケーリングレイヤーの1つです。序盤はNoobの購入とOofの生成に重点が置かれますが、中盤への移行にはルーンを効果的にロールし、レベルアップさせる方法を深く理解する必要があります。ルーンは、最初の大きな進行の壁にぶつかった際、標準的なショップのアップグレードを遥かに凌駕する乗算ボーナスを提供します。

| システム構成要素 | 主な機能 | 要件 |
| :--- | :--- | :--- |
| **ルーンのロール** | ランダムなステータスを持つ新しいルーンを獲得 | Oof / 進行トークン |
| **ルーンのレベル上げ** | 既存のルーンのパワーを強化 | 重複ルーン / 通貨 |
| **ルーンの幸運** | 高いレアリティが出る確率を上昇 | コミュニティサーバーへの参加 |
| **ルーンスピード** | ロール間の待ち時間を短縮 | グループ参加 + ゲームへの「いいね」 |

<Callout type="tip" title="進行のヒント">
序盤のNoob購入を無視しないでください。**Noob Incrementalのルーンレベル**の進行は、最初のロールに資金を供給するためにどれだけのOofを生成できるかに直結しています。
</Callout>

Ghoulax Gamesによって設計された公式のゲームプレイループでは、プレステージリセットを試みる前にこれらのボーナスを積み重ねることが推奨されています。早い段階でルーンの効率に焦点を当てることで、その後のすべての周回が前回よりも速く、より生産的になることが保証されます。

</Section>

<Section icon={Sparkles} title="Noob Incremental ルーンレベルの最適化方法">

**Noob Incrementalのルーンレベル**を上げることは、停滞したOof生産を打破するための主要な手段です。ルーンをレベルアップさせるとその基本倍率が増加し、それが所持しているNoobの総数やプレステージボーナスと相互作用します。これにより、高いプレステージレイヤーに到達するために不可欠な複利効果が生まれます。

**動画ハイライト：**
*(注：2026年6月現在、公式の開発者トレーラーは公開されていません。プレイヤーは視覚的なUIのヒントとして、ショップメニュー内のゲーム内チュートリアルパネルを参照してください。)*

<Steps>
  <Step num="1" title="Oofの基盤を確立する">
    ルーンメニューに触れる前に、安定したOofの流れを維持するのに十分なNoobを購入していることを確認してください。ルーンは、安定した土台を必要とする時間とリソースの投資です。
  </Step>
  <Step num="2" title="コミュニティブーストを有効にする">
    ボーナスが有効でない状態でルーンをロールしてはいけません。スピードブーストのためにGhoulax Gamesグループに参加し、幸運ブーストのためにDiscordに参加してください。これらは恒久的な+10%の修正値です。
  </Step>
  <Step num="3" title="ロールを開始する">
    UIのルーンセクションに移動します。蓄積した通貨を使用してロールを開始します。高度な最適化を心配する前に、まずは基本ルーンのフルセットを揃えることに集中しましょう。
  </Step>
  <Step num="4" title="主要なルーンをレベルアップさせる">
    重複したルーンや特定のレベルアップリソースを入手したら、最高のOofまたはプリズム倍率を提供するルーンに集中してください。レベルが高くなるとコストは減少（効率化）しますが、リターンは増加します。
  </Step>
</Steps>

| ルーンアクション | リソースコスト | 期待されるメリット |
| :--- | :--- | :--- |
| **基本ロール** | 低いOof / トークン | 新しいルーンスロットの解放 |
| **レベルアップ** | ルーンの欠片 / トークン | 倍率（%）の増加 |
| **リロール** | プレミアム通貨 / ポーション | ステータスの最適化 |

<Callout type="warning" title="効率に関する警告">
ルーンのレベル上げを急ぎすぎると、Oofの備蓄が枯渇し、不可欠なショップアップグレードが買えなくなる可能性があります。Noobとルーンレベルへの支出のバランスをとりましょう。
</Callout>

</Section>

<Section icon={Trophy} title="高度なレベル上げ戦略">

上級プレイヤーは、リソースを無駄にすることなく**Noob Incrementalのルーンレベル**を最適化するために、「一括ロール」と「ターゲットレベル上げ」に焦点を当てます。進行するにつれてレベル上げのコストは指数関数的に増加するため、より計算されたリソース管理アプローチが必要になります。

1.  **一括ロール**: 50回以上のロールが可能になるまでトークンを貯めてください。これにより、+10%の幸運ボーナスの恩恵を受ける統計的な可能性が高まります。
2.  **ターゲットレベル上げ**: 見つけたすべてのルーンをレベルアップさせないでください。第3プレステージレイヤーに到達するまでは、Oof獲得量またはルーンスピードを強化するルーンのみに集中しましょう。
3.  **ポーションのタイミング**: 通貨を大量に蓄えている時にのみ、ルーン幸運ポーションを使用してください。わずか5〜10回のロールのためにポーションを使用するのは、貴重なリソースの無駄です。

| レベル範囲 | 重点リソース | 戦略 |
| :--- | :--- | :--- |
| **レベル 1-10** | Oof / 基本トークン | 装備しているすべてのルーンをレベルアップ |
| **レベル 11-25** | 高度なトークン | Oof倍率に集中 |
| **レベル 26+** | プレステージ通貨 | プリズムとスピードに集中 |

アップデート情報については、公式のGhoulaxDev X (Twitter) アカウントをチェックしてください。時折、期間限定で基本幸運やレベルアップ成功率が上昇する「ルーンイベント」が開催されることがあります。
</Callout>
